# component-inspector

Public monorepo for the Itera component inspector SDK packages.

## Packages

- `@iteraai/inspector-protocol`
- `@iteraai/react-component-inspector`

## Local Development

Use the Node version in `.nvmrc`, then install and run the repo checks:

```bash
npm install
npm run build
npm run lint:ci
npm run type-check
npm run test
npm run test:pack
```
